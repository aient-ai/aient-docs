# Webhook event schemas

> Generated from `@aient/events`. Do not edit this page by hand.

Aient webhooks use CloudEvents 1.0 structured JSON. Only events marked public are tenant-selectable in webhook endpoint settings.

## Tenant-selectable events

- [ai.aient.instrumentation.pr_merged](./ai.aient.instrumentation.pr_merged) - An Aient instrumentation pull request was merged.

- [ai.aient.integration.github.connected](./ai.aient.integration.github.connected) - A GitHub installation became usable.
- [ai.aient.integration.github.disconnected](./ai.aient.integration.github.disconnected) - A GitHub installation became unusable.
- [ai.aient.integration.linear.connected](./ai.aient.integration.linear.connected) - A Linear workspace connection became usable.
- [ai.aient.integration.linear.disconnected](./ai.aient.integration.linear.disconnected) - A Linear workspace connection became unusable.
- [ai.aient.integration.mcp.connected](./ai.aient.integration.mcp.connected) - An MCP client connection became usable.
- [ai.aient.integration.scm.connected](./ai.aient.integration.scm.connected) - A source control provider connection became usable.
- [ai.aient.integration.scm.disconnected](./ai.aient.integration.scm.disconnected) - A source control provider connection became unusable.
- [ai.aient.integration.slack.connected](./ai.aient.integration.slack.connected) - A Slack workspace connection became usable.
- [ai.aient.integration.slack.disconnected](./ai.aient.integration.slack.disconnected) - A Slack workspace connection became unusable.
